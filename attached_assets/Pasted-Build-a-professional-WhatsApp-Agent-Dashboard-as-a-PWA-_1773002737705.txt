- Read all open escalations from `escalations` where `status = 'open'`
- Read full message history from `messages` by `customer_phone` ordered by `created_at ASC`

- Update `escalations` status to `'closed'` when conversation is closed

## Backend API Endpoints

### Auth
- `POST /api/login` — checks password against `DASHBOARD_PASSWORD` env var, sets session cookie

### Escalations
- `GET /api/escalations` — returns all open escalations ordered by `created_at DESC` (protected)
- `POST /api/escalate` — called by n8n when AI hands off a customer. Expects header `x-webhook-secret`. Body: `{ customer_phone, escalation_reason }`. Inserts into `escalations` table, triggers push notification to agent.
- `POST /api/close` — closes an escalation. Body: `{ customer_phone }`. Updates `escalations` status to `'closed'`, sends closing message via n8n webhook, triggers n8n to send WhatsApp message to customer. (protected)

### Messages
- `GET /api/messages/:phone` — returns full message history for a customer from `messages` table (protected)
- `POST /api/send` — agent sends a message. Body: `{ customer_phone, message }`. Saves to `messages` table with `sender = 'agent'`, calls `N8N_SEND_WEBHOOK` to deliver via WhatsApp. (protected)
- `POST /api/incoming` — called by n8n when customer sends a message during escalation. Expects header `x-webhook-secret`. Body: `{ customer_phone, message_text }`. Saves to `messages` table with `sender = 'customer'`, triggers push notification.

### Push Notifications
- `GET /api/push/vapid-public-key` — returns VAPID public key
- `POST /api/push/subscribe` — saves push subscription to memory
- `POST /api/push/unsubscribe` — removes push subscription
